# ZIO ← Cats Effect

**ZIO 2** for developers who know **Cats Effect 3**.

## Why Learn ZIO?

ZIO is a powerful effect system with built-in dependency injection and typed errors. While ZIO and Cats Effect share the same core principles, they differ in implementation details and API design.

### Key Differences

- **Environment:** ZIO has `ZIO[R, E, A]` with environment parameter for DI
- **Dependency Injection:** Uses `ZLayer` instead of Tagless Final
- **Error Handling:** Typed error channel with accumulation support
- **Streaming:** ZStream for high-performance push-based streaming
- **STM:** Built-in Software Transactional Memory (no separate library)
