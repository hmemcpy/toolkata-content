# Effect ← ZIO

**Effect** for developers who know **ZIO 2**.

## Why Learn Effect?

Effect is a powerful functional effect system for TypeScript that brings ZIO's principles to the JavaScript ecosystem. While Effect and ZIO share the same core design philosophy, they differ in parameter order, error handling semantics, and ecosystem integration.

### Key Differences

- **Parameter Order:** Effect is `Effect<A, E, R>` vs ZIO's `ZIO[-R, +E, +A]` — Success type comes first
- **Type System:** No variance annotations, no type aliases needed for readability
- **Error Channel:** Fatal defects (`Effect.die`) vs typed errors (`Effect.fail`)
- **Generators:** `Effect.gen` with `yield*` instead of for-comprehensions
- **Services:** `Context.Tag` class pattern vs `ZIO.service[T]`
